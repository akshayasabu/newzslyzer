╔══════════════════════════════════════════════════════════════════════╗
║  RECOMMENDATION ALGORITHM                                            ║
╚══════════════════════════════════════════════════════════════════════╝

  Priority Levels:
  ─────────────────────────────────────────────────────────────────────
  1. Manual Interests (HIGHEST)    → 75 points (15x weight)
  2. Reading History (MEDIUM)      → 5-10 points per read
  3. Deep Reading Bonus (MEDIUM)   → 2x multiplier (30+ seconds)
  4. Search Keywords (MEDIUM)      → 15-30 points
  5. Novelty Filter                → Exclude read articles

  Example Scoring:
  ─────────────────────────────────────────────────────────────────────
  User: Manual interests = Sports, Tech
        Reading history = Sports (5 reads)
        Recent search = "cricket"

  Sports article about cricket:
    75 (manual) + 25 (reading) + 30 (search) = 130 points ⭐

  Tech article:
    75 (manual) + 0 (reading) + 0 (search) = 75 points

  Politics article:
    0 (manual) + 10 (reading) + 0 (search) = 10 points

  Result: Sports article recommended first!

║  COMBINATION LOGIC                                                   ║
╚══════════════════════════════════════════════════════════════════════╝

  The system intelligently combines:
  ─────────────────────────────────────────────────────────────────────
  1. Manual Interests (User-selected)     → Highest priority
  2. Reading History (Automatic tracking) → Medium priority
  3. Deep Reading (Time spent)            → Bonus multiplier
  4. Search Keywords (Recent searches)    → Relevance boost

  Example:
  ─────────────────────────────────────────────────────────────────────
  User Profile:
    Manual: Sports, Tech
    Auto:   Sports (5 reads), Politics (2 reads)

  Combined Preferences:
    1. Sports    → Manual + Auto (HIGHEST)
    2. Tech      → Manual only (HIGH)
    3. Politics  → Auto only (MEDIUM)

  Result: Best of both worlds!
